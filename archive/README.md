# Archive

Esta pasta concentra artefatos historicos e materiais fora do runtime ativo da plataforma.

## Escopo
- `olds/`: snapshots e pacotes zip de versoes anteriores.
- `host.py`: servidor Python legado usado em ciclos antigos de desenvolvimento local.
- `GardenQuest Projeto Detalhado.html`: export historico de planejamento/conversa.
- `legacy/`: placeholder para futuros materiais legados que precisem sair da raiz.

## Regras de uso
- Conteudo em `archive/` nao deve ser usado como dependencia de runtime.
- Nao referenciar arquivos deste diretorio em scripts de build/deploy.
- Se algum item precisar voltar para o fluxo ativo, mover explicitamente para a arvore principal e atualizar a documentacao correspondente.
