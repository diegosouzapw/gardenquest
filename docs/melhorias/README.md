# Melhorias — Guia Rápido

Este diretorio contem as tasks individuais do plano de melhorias.

## Estrutura
- `fase-01-critico/`
- `fase-02-importante/`
- `fase-03-moderado/`
- `fase-04-evolucao/`

## Como usar
1. Escolha a task no `docs/CHECKLIST_EXECUTAVEL_POR_PR.md`.
2. Abra o arquivo da task na fase correspondente.
3. Execute os passos tecnicos na ordem dos arquivos listados.
4. Rode as validacoes e atualize `docs/melhorias/STATUS_EXECUCAO.md`.
